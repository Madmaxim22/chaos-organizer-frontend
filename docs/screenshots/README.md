# Скриншоты для документации

Добавьте сюда скриншоты приложения для README.md в корне проекта.

Рекомендуемые файлы:

| Файл | Что показать |
|------|----------------|
| `01-glavnoe-okno.png` | Главный экран: список сообщений, сайдбар слева, панель поиска, поле ввода |
| `02-sidebar-kategorii.png` | Боковое меню с категориями (Все, Изображения, Видео, Аудио, Файлы, Ссылки, Избранное) и кнопками Экспорт/Импорт |
| `03-panel-vvoda.png` | Панель ввода: текст, эмодзи, бургер-меню (файл, аудио, видео, геолокация, напоминание, шифрование) |
| `04-palitra-komand.png` | Палитра команд при вводе «@»: команды @chaos: и @schedule: |
| `05-zakreplenie-izbrannoe.png` | Закреплённое сообщение сверху и сообщение в избранном (звёздочка) |
| `06-poisk.png` | Поиск: поле ввода и фильтры по типам (текст, ссылки, изображения и т.д.) |
| `07-napominanie.png` | Модальное окно создания напоминания или сообщение с командой @schedule: |
| `08-shifrovanie.png` | Режим шифрования (замок) и расшифрованное сообщение по паролю |

Сделайте скриншоты в рабочем приложении и сохраните в эту папку с указанными именами.
