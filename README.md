# Chaos Organizer

[![GitHub Pages](https://img.shields.io/badge/GitHub_Pages-View_demo-222?style=flat-square&logo=github)](https://madmaxim22.github.io/chaos-organizer-frontend/)
[![Render](https://img.shields.io/badge/Render-Backend_deployed-46E3B7?style=flat-square&logo=render)](https://chaos-organizer-backend-q24c.onrender.com)

Органайзер в формате мессенджера: вы отправляете боту сообщения, файлы и команды — он хранит, ищет, напоминает и синхронизирует данные между вкладками.

**Полная документация для пользователей и схемы работы функций** перенесены во фронтенд: [chaos-organizer-frontend/docs/](chaos-organizer-frontend/docs/) — их можно открыть из приложения по кнопке «Документация» в боковом меню или напрямую через `docs/index.html` после сборки.
